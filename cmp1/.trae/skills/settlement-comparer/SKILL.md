# Settlement Comparer

## 功能
- 读取 `附件五：中国铁塔云南公司2024-2025年土建杆塔施工集中采购项目施工模块安全生产费明细表.xlsx` 文件
- 提取 `{模块名称，红河/文山}` 列所对应的数据
- 与 `{施工内容，单项结算金额单价}` 作比较
- 根据比较结果进行颜色标记：
  - 如果 `{施工内容，单项结算金额单价}` 相同，在新生成的表里将 `{单项结算金额单价}` 标绿色
  - 如果不一致，将数据标红
  - 如果不存在的，添加到新生成的表的末尾，并且将 `{单项结算金额单价}` 置为黄色

## 使用方式
在终端中运行以下命令：
```bash
python .trae/skills/settlement-comparer/compare_settlement.py
```

## 输入文件
- `附件五：中国铁塔云南公司2024-2025年土建杆塔施工集中采购项目施工模块安全生产费明细表.xlsx`
- 之前生成的包含施工内容和单项结算金额单价的文件

## 输出文件
- 生成带有颜色标记的新Excel文件
