# OpenClaw 结算计算器

## 功能说明

本skill用于计算.xlsx文件中的结算金额，按照以下规则进行计算：

### 计算规则

1. **单项结算金额** = 施工量 * 单项结算金额
2. **工程结算金额31%** = 单项结算金额 * 0.31
3. **外包结算比例** = 70%
4. **外包结算单价** = 单项结算单价 * 外包结算比例
5. **外包结算金额** = 施工量 * 外包结算单价
6. **外包结算金额31%** = 外包结算金额 * 0.31
7. **本次外包结算比例** = 90.00%
8. **本次外包结算金额** = 外包结算金额30% * 本次外包结算比例
9. **外包已结算金额** = 0.00
10. **外包结算剩余金额** = 外包结算金额31% - 本次外包结算金额 - 外包已结算金额
11. **利润率比例** = (工程结算金额31% - 外包结算金额31%) / 工程结算金额31%

## 使用方法

1. 将需要计算的.xlsx文件放在当前目录下
2. 调用本skill
3. 系统会自动处理所有.xlsx文件，并生成计算结果

## 处理流程

1. 扫描当前目录下所有以final.xlsx结尾的文件
2. 对每个文件，读取数据并按照计算规则进行计算
3. 在每个文件中添加相应的计算列
4. 保存计算结果到新文件（原文件名后添加_calculated）

## 注意事项

- 确保.xlsx文件中包含"施工量"、"单项结算金额"和"单项结算单价"列
- 计算结果会保存为新文件，不会覆盖原文件
- 支持处理多个.xlsx文件
